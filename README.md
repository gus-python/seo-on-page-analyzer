# SEO On-Page Analyzer

Ferramenta simples desenvolvida em **Python** para análise de fatores básicos de **SEO on-page**.

O projeto foi criado com o objetivo de automatizar verificações iniciais de SEO em páginas web, avaliando elementos importantes para otimização em mecanismos de busca.

## Funcionalidades

O analisador verifica alguns fatores básicos de SEO, como:

- Presença da **tag H1**
- Presença da **palavra-chave no H1**
- Palavra-chave no **início do H1**
- Presença da palavra-chave no **título**
- Palavra-chave no início do **title**
- Tamanho do **title**
- Presença da palavra-chave na **meta descrição**
- Tamanho da **meta descrição**
- **Pontuação SEO** baseada nos critérios analisados

## Tecnologias utilizadas

- Python
- Requests
- BeautifulSoup

## Como usar

1. Execute o script Python.
2. Insira a **palavra-chave principal** quando solicitado.
3. Informe as **URLs ou textos** que deseja analisar.
4. O programa retornará os resultados da análise e a pontuação de SEO.
